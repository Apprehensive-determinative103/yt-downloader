## 📌 About this project

This application allows you to save videos and audio tracks from various internet services to your local storage. It serves as a visual interface for the powerful yt-dlp tool. You use this software to store content for offline viewing or listening. The interface simplifies the process of extracting media files without the need for command-line instructions. 

It handles various formats, including high-definition video and high-quality audio files. You can download single videos or entire playlists with a few clicks. The tool manages the conversion process automatically, ensuring your files work on your Windows computer or mobile devices.

## 🖱️ How to use the interface

Once you open the software, you see a clean window with a primary text field. Follow this process to save your first file:

1. Copy the address of the video or playlist from your web browser.
2. Paste the address into the box at the top of the application window.
3. Select your preferred output format from the dropdown menu, such as MP4 for video or MP3 for audio.
4. Choose the resolution, such as 4K, 1080p, or 720p, depending on your needs.
5. Pick a folder on your computer where the program should save the file.
6. Click the button labeled Download to begin the task.

The status bar at the bottom of the window shows the progress. The application notifies you when the process finishes.

## 📝 Troubleshooting common issues

If you encounter difficulties, review these common scenarios to resolve them quickly.

* The download does not start: Check your active internet connection. Ensure you copied the full web address.
* The file does not play: Ensure you have a media player installed that supports the chosen file format. Standard players usually handle all common outputs.
* Slow download speeds: This often depends on the source website and your network bandwidth. Close other applications that use heavy data during your download.
* File is missing: Click on the Folder icon in the application to open the directory where your saved files reside.
